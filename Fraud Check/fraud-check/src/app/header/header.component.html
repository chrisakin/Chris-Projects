FRAUD CHECK

This is a Public safety solution Web Application called Fraudcheck. It helps reduce corruption by checking for fraudulent people and practices in Nigeria so as to reduce the rate at which people are defrauded of their money. It also gives an Artificial Intelligence (AI) based platform for users to interact with the Web Application better.
